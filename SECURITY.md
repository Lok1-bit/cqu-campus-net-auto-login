# 安全政策

## 支持版本

安全修复仅面向最新的 `main` 分支和最新版本标签。当前支持版本为 `0.1.x`。

## 私下报告漏洞

请使用 GitHub 的 [Private vulnerability reporting](https://github.com/Lok1-bit/cqu-campus-net-auto-login/security/advisories/new) 私下报告可能导致凭据泄露、向非预期主机发送认证信息、命令执行或权限提升的问题。

如果页面没有显示私下报告入口，请先通过维护者 [GitHub 主页](https://github.com/Lok1-bit) 建立联系，询问私密联系方式。不要在公开 Issue 中披露漏洞细节。

报告中请包含：

- 受影响的提交或版本；
- 不含真实凭据的最小复现步骤；
- 预期影响和可能的缓解方案；
- 是否已经公开披露。

## 隐私要求

无论公开还是私下报告，都不要发送真实密码。公开 Issue、Pull Request、评论和附件中也不得包含：

- 校园网账号或统一认证号；
- 完整 IPv4/IPv6 或 MAC 地址；
- 含 `userip`、`usermac` 等参数的原始门户 URL；
- 未脱敏日志、网络抓包或截图。

维护者会尽快确认报告并沟通处理进展，但本个人项目不提供固定响应时限或服务等级承诺。
