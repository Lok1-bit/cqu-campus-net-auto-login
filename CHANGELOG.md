## [0.1.0] - 2026-07-04

### Added

- 重庆大学 Dr.COM 在线状态检测与掉线自动登录。
- Windows 登录后静默启动的计划任务安装流程。
- Windows 凭据管理器存储，不使用明文密码配置。
- 可配置检查周期、失败退避、请求超时和日志上限。
- 日志脱敏、滚动和单实例后台监控。
- 离线 Pester 测试与只读状态诊断。
- MIT License、贡献指南、安全政策和 GitHub 协作模板。
